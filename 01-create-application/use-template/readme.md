#### 使用模板创建 electron 应用

使用模板创建主要是通过 git 工具来拉取 一个预先创建好并发布到网络上的项目

依赖：git，npm

##### 步骤

1. 拉取项目

   ```shell
   git clone https://github.com/electron/electron-quick-start
   ```

2. 进入项目目录

   ```shell
   cd electron-quick-start
   ```

3. 安装项目依赖

   ```shell
   npm install
   ```

4. 启动项目

   ```shell
   npm start
   ```
